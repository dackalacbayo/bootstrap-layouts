Header variations

We waited until we could do it right. Then we did! Instead of creating a carbon copy.

  • Solid color

    Use only one color on your header and display some text on it.
  • Gradient color

    You can create a gradient background with two or more colors.
  • Image

    Display an image as a background image with an optional overlay.
  • Particle

    Display hundreds of dots moving in your header with less attention.
